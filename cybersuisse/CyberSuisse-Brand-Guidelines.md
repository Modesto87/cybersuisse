## 🔤 Typography

### Primary Font (Headings & Titles)
**Font Family:** `Montserrat`
**Usage:** Main Titles, Section Headers, Subtitles, Navigation
**Weights:**
- **Black (900)**: Hero Headlines ("STOP AUX CYBERATTAQUES")
- **Bold (700)**: Section Titles
- **SemiBold (600)**: Card Titles

### Secondary Font (Body Text)
**Font Family:** `Inter`
**Usage:** Paragraphs, Descriptions, Captions, Quotes
**Weights:**
- **Regular (400)**: Standard Body Text
- **Medium (500)**: UI Elements, Buttons, Captions

## 🎯 Usage Rules

1.  **Blue = Base** → Use for backgrounds and trust elements.
2.  **Gold = Action** → Use for CTA buttons and elements requiring attention.
3.  **White = Readability** → Use for text on dark backgrounds.
4.  **Red = Urgency** → Use sparingly, only for alerts.
5.  **Green = Success** → Use for confirmations and positive status.
